trait IterableProxy[+A]
extends Iterable[A] with Proxy
This class implements a proxy for iterable objects. It forwards all calls to a different iterable object.
Author
Matthias Zenger
Martin Odersky
Version
2.0, 31/12/2006
Direct Known Subclasses:
